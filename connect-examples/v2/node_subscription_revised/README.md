# Node Subscriptions


## Getting started
```
$ cd server
$ npm i 
$ cp .env.example .env
```

Change the value of `SQ_ACCESS_TOKEN` to your sandbox access token from [Square](developer.squareup.com)

```
$ npm run seed
$ npm run dev
```

Open another Terminal

```
$ cd app
$ npm i
$ npm run start
```

Use the app to create subscriptions
